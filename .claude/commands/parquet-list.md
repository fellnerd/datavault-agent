# Parquet-Dateien auflisten

Listet alle Parquet-Dateien in einem ADLS Gen2 Verzeichnis auf.

## Parameter

- `folder_path` (required): Pfad zum Ordner in ADLS

## Beispiele

```
Zeige alle Parquet-Dateien in jira/sql
```

```
Liste die Dateien im Ordner werkportal/postgres
```

## Bekannte Ordner

- `jira/sql` - Jira Agile, Platform, ServiceDesk Daten
- `werkportal/postgres` - Werkportal PostgreSQL Export
